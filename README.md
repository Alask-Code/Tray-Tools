# Atalhos do Windows na Tray

## Atalhos
- Microsoft Edge
   - Nova Aba
   - Nova Aba InPrivate
- Visual Studio Code
   - Abrir o programa
   - Abrir uma Pasta
- Command Prompt
   - Nova Guia do Terminal

## Stack Utilizada
 - ElectronJS
    - Tray Manager
 - Electron Reload
    - Suporta live reload em nível de aplicação
 - Electron Builder
    - Compilação para múltiplas plataformas
